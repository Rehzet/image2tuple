# image2tuple
This script processes images and returns a tuple array that can be used with Pillow's `ImageDraw.point()` to recreate the image. 

I created this script to display images on my 0.96-inch SSD1306 OLED display.


# Python Dependencies
- argparse >= 1.4.0
- Pillow >= 2.1.1
- numpy >= 10.4.0

# Examples

`python3 img2tuple.py --path "/home/rehzet/Im√°genes/Logos/fujitsu.png" -W 100 -k -s`
